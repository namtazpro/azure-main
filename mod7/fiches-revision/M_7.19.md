> Rappel des niveaux : **1** = familiarisation · **2** = connaissance générale + application · **3** = connaissance détaillée + analyse/correction.
> Le niveau **2** en fait un module important : il faut savoir appliquer les inspections après événement anormal et décider si l’aéronef peut être remis en vol.

## 🎯 À retenir en priorité
- Les événements anormaux déclenchent des inspections non programmées, généralement via TLB/CRM après compte rendu équipage.
- La gravité dépend de la masse, vitesse, direction des charges, météo et configuration.
- Les inspections se font par phases : phase 1 extérieure, phase 2 détaillée interne, phase 3 avec dépose d’organes majeurs si nécessaire.
- Foudroiement et HIRF peuvent produire des dommages visibles, cachés ou électroniques transitoires.
- Atterrissage dur, surcharge et turbulence sévère exigent collecte des paramètres équipage/FDR avant inspection.

## Synthèse du cours

## Introduction — Généralités sur les événements anormaux

Un **événement anormal** est une situation sortant de l’exploitation normale et pouvant affecter la structure, les systèmes ou la navigabilité.

Exemples :
- grippage frein ;
- pollution mercure ;
- surpressurisation cabine ;
- foudroiement ;
- dépassement VMO/VLE/VFE ;
- impact oiseau ou grêle ;
- atterrissage dur ou en surcharge ;
- éclatement pneumatique ;
- freinage d’urgence/surchauffe ;
- tail strike ;
- sortie de piste ;
- vol dans cendres volcaniques ;
- panne moteur ou coupure moteur en vol.

Les inspections sont déclenchées par le compte rendu équipage dans le **Technical Log Book (TLB)** ou **Compte Rendu Matériel (CRM)**.

Séquence typique :
- découper l’aéronef en zones majeures : fuselage, voilure, mâts moteurs, empennages ;
- subdiviser en sous-zones ;
- appliquer des phases :
  - **phase 1** : inspection générale externe ;
  - **phase 2** : inspection détaillée interne avec accès/dépose ;
  - **phase 3** : inspection très détaillée avec dépose d’organes majeurs.

Règle :
- si phase 1 sans défaut : phases 2/3 non nécessaires ;
- si phase 1 avec défaut : phase 2 obligatoire ;
- si phase 2 avec défaut : phase 3 obligatoire.

## 1. Inspection suite à foudroiement

### 1.1 Effets de la foudre

La structure métallique extérieure forme un bouclier protecteur. La foudre touche généralement l’aéronef en deux points :
- point d’entrée ;
- point de sortie.

Zones :
- **zone 1** : régions d’impact où les décharges entrent/sortent ;
- **zone 2** : surfaces d’écoulement pouvant aussi être impactées ;
- **zone 3** : surfaces de transfert du courant.

Zones 1/2 typiques :
- radôme ;
- nacelles ;
- extrémités ailes, stabilisateurs, gouvernes ;
- volets de bord d’attaque/fuite ;
- fuselage/protubérances ;
- trains sortis ;
- pitots, antennes, drains eaux usées.

### 1.2 Effets directs et indirects

Sur structures métalliques :
- points de soudure ;
- perforations ;
- peau ou fixations brûlées/décolorées.

Effets indirects :
- champ électromagnétique en zone non pressurisée ;
- défauts transitoires ou permanents sur systèmes électriques/électroniques ;
- indications erronées, écrans coupés, perturbations navigation/communication.

Sur composites :
- conductivité assurée par éléments conducteurs intégrés ;
- impacts parfois invisibles ;
- dommages internes possibles ;
- arcs/échauffements sur fixations et supports.

Nota : les **déperditeurs statiques** et tresses de métallisation évacuent l’électricité statique et limitent les interférences radio ; ils n’empêchent pas la foudre.

### 1.3 Dégâts possibles

- roulements de gouvernes brûlés ;
- gouverne bloquée ;
- générateurs AC, distribution, contacteurs, valves carburant endommagés ;
- composants sur lignes d’antenne hors service ;
- magnétisation de structure perturbant navigation, communication ou jaugeage.

Cas hélices :
- hélice souvent point d’entrée ;
- extrémité de pale noircie ou détériorée ;
- hélice à réviser ou remplacer ;
- régulateurs hélice concernés à remplacer ou réviser ;
- réparation selon procédure hélicier par spécialistes qualifiés.

### 1.4 Inspection et mise à la terre

L’inspection est déclenchée par rapport équipage, mais l’équipage peut ne pas détecter le foudroiement : les visites avant/après vol sont donc essentielles.

La **mise à la terre** :
- permet l’écoulement des charges statiques ;
- protège le personnel contre défauts d’isolement ;
- est obligatoire pour maintenance/mise en œuvre, notamment avitaillement ;
- câble fixé d’abord au point sol, puis à l’aéronef.

## 2. Inspection suite à pénétration HIRF

### 2.1 Définition et sources

Les **HIRF (High Intensity Radiated Fields)** sont des ondes électromagnétiques de forte puissance issues de sources au sol ou embarquées.

Sources :
- radars civils/militaires ;
- émetteurs radio AM/FM ;
- relais GSM/4G/5G ;
- radars approche/tour ;
- systèmes défense aérienne ;
- radars météo d’autres aéronefs ;
- systèmes militaires ou guerre électronique.

Situations à risque :
- approche/décollage près radars puissants ;
- survol bases militaires ;
- proximité radiodiffusion haute puissance ;
- parking près radars/antennes relais.

Différence : le foudroiement est violent et ponctuel ; l’exposition HIRF peut être continue, progressive et sans signe extérieur.

### 2.2 Principe physique et fréquences

Les ondes induisent des courants parasites dans câblages et structures par couplage inductif ou capacitif.

Champ électrique simplifié :

`E (V/m) = √(30 × P × G) / d`

Exemple : radar 100 kW, gain 1000, distance 500 m → environ **346 V/m**.

Bandes critiques :

| Bande | Fréquence | Application | Niveau typique |
|---|---|---|---|
| VHF | 30–300 MHz | communications, FM | 10–100 V/m |
| UHF | 300 MHz–3 GHz | radar, télécoms, GSM | 50–200 V/m |
| Micro-ondes | 3–30 GHz | radar météo, satellite | 100–600 V/m |

Seuils certification HIRF DO-160 section 20 :
- **200 V/m** pour VHF/UHF usage général ;
- **400 V/m** pour environnements sévères.

Loi inverse du carré :

`E₂ = E₁ × (d₁ / d₂)²`

Exemple : 400 V/m à 200 m → 100 V/m à 400 m.

### 2.3 Effets avioniques

Perturbations transitoires :
- perte momentanée écran ;
- déconnexion pilote automatique ;
- parasites radio ;
- messages d’erreur sans défaut réel.

Perturbations permanentes :
- composants électroniques défaillants ;
- mémoire corrompue ;
- destruction circuits par surtension.

Systèmes sensibles :
- FMS : navigation, plan de vol ;
- EFIS : PFD/ND ;
- pilote automatique ;
- TCAS ;
- VHF/HF ;
- transpondeur ;
- GPS.

### 2.4 Zones sensibles et protections

Zones critiques :
- radôme ;
- cockpit et baie avionique ;
- soutes avioniques ;
- antennes externes.

Zones modérées : fuselage, voilure, empennages, moteurs.

Protections :
- blindage tresse métallique : couverture minimale **85 %**, connexion 360° aux connecteurs ;
- blindage feuillard : protection **> 95 %**, câbles critiques ;
- filtres EMI/RFI : bloquent fréquences parasites **> 1 MHz**, laissent passer signaux utiles **< 1 kHz** ;
- mise à la masse/bonding.

Résistances maximales :
- entre structures : **< 0,005 Ω** ;
- blindage vers structure : **< 0,01 Ω**.

Normes : DO-160 section 20, CS-25 / FAR Part 25.

### 2.5 Conditions et procédure d’inspection HIRF

Déclenchement :
- déconnexions AP ;
- messages d’erreur sans défaut confirmé ;
- affichages erronés ;
- parasites radio.

Informations à recueillir :
- phase de vol, position géographique ;
- symptômes permanents/intermittents ;
- disparition après éloignement ;
- systèmes touchés simultanément ;
- FDR : horodatage, déconnexions AP, position GPS.

Indices HIRF vs défaut réel :

| HIRF | Défaut réel |
|---|---|
| symptômes approche/décollage | symptômes permanents |
| disparition après éloignement | un seul système affecté |
| plusieurs systèmes affectés | codes défauts cohérents |
| aucun code défaut | tests révèlent anomalie |
| tests sol OK | anomalie reproductible |

Équipements :
- ohmmètre basse impédance ;
- multimètre ;
- mégohmmètre **500 VDC** ;
- outillage standard.

Contrôles :
- connecteurs : surchauffe, broches, corrosion, verrouillage ;
- blindage-masse : **R < 0,01 Ω** ;
- coax radôme : isolation âme/blindage **R > 10 MΩ** ;
- bonding structure : **R < 0,005 Ω**.

Tests fonctionnels :
- BITE FMS, EFIS, AP/FD, radios, transpondeur, TCAS, GPS ;
- continuité bout à bout **R < 1 Ω** ;
- isolation **R > 10 MΩ** avec mégohmmètre 500 VDC ;
- essai statique ou vol test **30 min** selon besoin.

Critères remise en service :
- BITE PASS ;
- aucun message erreur ;
- affichages normaux ;
- mesures dans tolérances.

## 3. Atterrissage lourd ou en surcharge

### 3.1 Données à collecter

Avant AMM :
- équipage : atterrissage lourd, en ligne droite, en dérapage, bruit suspect ;
- FDR : accélération verticale, vitesse verticale ft/min, masse avion.

Facteur de charge :

`n = a / g`, avec `g = 9,81 m/s²`

Exemple : 19,62 m/s² → **2,0 g**.

Limites du cours :

| Type avion | n max atterrissage | n limite structurale |
|---|---:|---:|
| Avion ligne B737/A320 | 2,0–2,5 g | 3,75–4,0 g |
| Avion régional | 2,0–2,2 g | 3,5–3,8 g |

### 3.2 Généralités

L’inspection est principalement visuelle ; si nécessaire, appliquer les procédures NDTM.

Lors de l’ouverture d’accès/dépose portes :
- alignement ;
- absence de voile/crique ;
- peinture non endommagée ;
- fixations en état.

Inspection avant remise en vol. Pour train rentrant et incident train, mettre sur vérins par sécurité avant inspection/maintenance ; vérifier les trois trains sortis/verrouillés avant remise sur roues.

### 3.3 Inspection premier niveau

Objectif : dommages visibles extérieurs et défaillances structurelles.

Zones :
- trains : pneus, jantes, amortisseurs, tuyauteries freins, verrous, contrefiches, vérins, attaches ;
- nacelles : revêtements, capotages, commandes moteur, câblages, tuyauteries, silent-blocs, pales hélices ;
- voilure centrale : intrados/extrados, rivets, câblages, vérins, caissons ;
- voilures extérieures : revêtement, portes de visite, réservoirs, câblages ;
- fuselage avant : revêtement, logement train, radar, électronique ;
- fuselage central/arrière : hublots, empennage, gouvernes ;
- essai train.

Attention : une déformation extérieure jugée négligeable peut révéler un dommage interne.

### 3.4 Inspection deuxième niveau

Déclenchée si défauts au premier niveau. Inspecter structure environnante et support.

Exemples :
- avion sur vérins, déplacement amortisseurs ;
- déposer pneus, inspecter intérieur ;
- jantes désassemblées ;
- axe de roue à la loupe **grossissement 10** ;
- en doute : dye check ou magnétoscopie ;
- boulons train/contrefiches déposés, remplacés ou magnétoscopés ;
- alésages cellule : criques/élongation ;
- réservoirs : fuites/dommages ;
- planchers déposés ;
- circuits, câblages, volets, commandes essayés.

Réparation : adaptée aux défauts. Documents de bord renseignés avec dommages et suites.

## 4. Vol en turbulence

### 4.1 Déclenchement

Inspection principalement structurelle après :
- facteurs de charge excessifs ;
- manœuvres hors limites ;
- dépassement vitesse ;
- décrochage.

Limites opérationnelles du cours :

| Turbulence | n min | n max |
|---|---:|---:|
| Légère | +0,5 g | +1,5 g |
| Modérée | 0 g | +2,0 g |
| Sévère | -1,0 g | +2,5 g |
| Limite structurale | -1,0 g | +3,75 g |

Classification OACI :
- légère : tension légère ceintures ;
- modérée : occupants fermement pressés, marche difficile ;
- sévère : occupants violemment projetés, avion hors contrôle, variations altitude > **100 ft**.

### 4.2 Conditions d’inspection

Informations équipage :
- facteur de charge ;
- alarme VMO/MMO ;
- bruit structure ;
- vibrations/buffeting.

FDR :
- accélération verticale ;
- vitesse verticale ;
- VMO/MMO ;
- masse avion.

En cas de doute, recouper avec enregistreur de vol. Exemple Boeing 777 : facteur de charge entre **2,5 g et -1,0 g**, vitesse maxi opérationnelle + **20 kt VMO** ou **0,05 Mach MMO**.

### 4.3 Inspections

Inspection avant remise en vol si accélérations importantes en turbulence sévère.

Premier niveau :
- attitude normale ;
- nacelles, ailes, fuselage en configuration normale ;
- voilure centrale/extérieure : criques, déformations, rivets, circuits, caissons ;
- nacelles : capots, supports moteurs, commandes ;
- fuselage : revêtement, pare-brise, équipements, antennes ;
- empennage : gouvernes, jonctions, libre débattement.

Deuxième niveau :
- déposer planchers/accès ;
- inspecter longerons, caissons, attaches moteur ;
- essais circuits/câblages/volets/commandes ;
- inspecter réservoirs si dommage ;
- examiner équipements, conduits, poulies, câbles, empennage, trim.

Réparation selon défauts, documents de bord renseignés.

## ✅ Points clés à mémoriser
- Tout événement anormal doit être reporté TLB/CRM et traité selon AMM/ATA 05.
- Phase 1 OK → pas de phase 2/3 ; défaut en phase 1 → phase 2 ; défaut en phase 2 → phase 3.
- Foudre : rechercher entrée/sortie, brûlures, perforations, roulements/gouvernes, systèmes électriques.
- Hélice foudroyée : révision générale ou remplacement, régulateurs à remplacer/réviser.
- HIRF : peut ne laisser aucune trace extérieure ; rechercher symptômes transitoires multi-systèmes.
- Bonding HIRF : structures <0,005 Ω ; blindage vers structure <0,01 Ω ; isolation coax >10 MΩ.
- HIRF : VHF 30–300 MHz, UHF 300 MHz–3 GHz, micro-ondes 3–30 GHz.
- FDR après atterrissage dur : g, vitesse verticale, masse.
- Facteur de charge : `n = a/g`, g = 9,81 m/s².
- Inspection atterrissage dur : trains, nacelles, voilures, fuselage, essais train.
- Turbulence sévère : occupants projetés, avion hors contrôle, variations altitude >100 ft.
- Boeing 777 exemple : 2,5 g / -1,0 g, VMO +20 kt ou MMO +0,05 Mach.
