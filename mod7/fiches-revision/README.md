# Fiches de révision — Module 7 (Part-66 B1/B2)

Fiches de synthèse pour la révision du **Module 7 – Procédures d'entretien** de la licence Part-66.
Chaque fiche est issue **uniquement** du contenu des livrets de cours AIR FORMATION CFM 66 correspondants.

## Niveaux d'importance (syllabus Part-66)

- **1** = familiarisation (notions de base)
- **2** = connaissance générale + capacité d'application
- **3** = connaissance détaillée + analyse / action corrective

> Plus le niveau est élevé, plus le chapitre est important à maîtriser pour l'examen.
